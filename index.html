Chrome or Edge on Windows 10+, macOS, Android. Connects directly to WHOOP's BLE Heart Rate broadcast — no Python, no bridge.

  1. Open the WHOOP app → Device Settings → HR Broadcast → ON
  2. Click Connect WHOOP in the toolbar above
  3. Select your WHOOP from the browser's Bluetooth picker

Requires HTTPS or localhost · Chrome / Edge / Opera only · Firefox & Safari not supported

Bridge fallback Firefox / Safari

Run a small Python script that reads WHOOP over BLE and forwards data via WebSocket to any browser.

  1. Install: pip install bleak websockets
  2. Run: python bridge.py
  3. Enter the WebSocket address below and click Connect
